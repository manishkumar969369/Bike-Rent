# 🚴 Forecast-A-Ride: Bike Demand Prediction

## 📌 Problem

Predict daily bike-sharing demand using weather and calendar data.

## 🎯 Solution

Built a Linear Regression model with feature engineering and deployed using Streamlit.

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

## 📊 Features Used

temp, atemp, humidity, windspeed, season, weathersit, year, month, workingday

## 📈 Results

* Train RMSE: 0.38
* Test RMSE: 0.54

## 🚀 How to Run

```bash
streamlit run app.py
```
